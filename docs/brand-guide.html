Logo usage

The one rule that matters: never pair the boxed app icon with loose text.
✓ Do

Show the SS monogram solo — no box, no text beside it. This is how headers and avatars use the mark.

✓ Do

Use a full lockup (wordmark or script) when you need the name and mark together — they're designed as one locked unit.

✕ Don't

(F)

Voice

Warm, plain-spoken, a little nostalgic — a note on the kitchen table.
Do
  • Talk like family: "New one's up," not "A new episode has been published."
  • Let the serif headings carry the warmth; keep body copy simple.
  • Name things plainly — Listen, Watch, Request access.
  • Use color as a small touch, not a flood.
Don't
  • Sound corporate or salesy ("Subscribe for exclusive content!").
  • Over-explain or apologize in error messages.
  • Pair the boxed app icon with loose text.
  • Fill a screen with competing accent colors.
